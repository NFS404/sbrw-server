# Soapbox Race World Server

## Description

> This is the soap-box race world server developed in NodeJS.

## Requirements

> NodeJS (v10 or above)

> NPM (v6 or above)

> MongoDB server (you can use third party remote servers too.)

## Installation and Running

> npm i

> npm start

## What's done

> API List.

> Auth.

> Started driver persona.
